I - Lancer le programme Java
	Depuis une console, aller dans le dossier "Version Java complète".
	Lancer la commande "java -jar graph_astar_pierre_casati.jar"

	Les sources sont consultables dans "src".

II - Visualiser la solution c++ incomplète [UNIQUEMENT SUR WINDOWS]
	Aller dans le dossier "Version C++ incomplète\Debug".
	Lancer "GameOfLife.exe".

	Les sources sont consultables dans "Version C++ incomplète\GameOfLife\".
